fh2ad.util.filter_maps(maps, needed_layer, last_maps, last_cats=None, check_categories=True)¶

Keep only valid maps:

  • Correct layer

  • Not in the last X played

  • Not the same category as the last Y played (except theme is active)

  • If server full, supports 100 ppl (except theme is active)
