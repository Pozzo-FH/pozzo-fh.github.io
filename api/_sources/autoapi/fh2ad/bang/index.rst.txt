:mod:`fh2ad.bang`
=================

.. py:module:: fh2ad.bang

.. autoapi-nested-parse::

   !Bang commands, to run in the game chat

   All functions callable from ingame chat commands need to follow the pattern
   `func(invoker, words)`, where `words` is a `list(str)`, and `invoker` a
   `fh2ad.core.FHPlayer` instance.
