class fh2ad_plug.SteamRoller(shuffle_on_start=SHUFFLE_ON_START, swap_on_start=SWAP_ON_START, protect_commander=PROTECT_COMMANDER, protect_squadmembers=PROTECT_SQUADMEMBERS, protect_squadleader=PROTECT_SQUADLEADER)¶

Autobalance teams and swap/shuffle on round start

Note that the protect_X options have nothing to do with teamswap/-shuffle and are ignored for those purposes. they only are used when a player dies during the round

Note to self:

Overall Game State Change Pattern

  1. StatusChange (to “PreGame”)

  2. Reset (doesn’t happen in the first round after a server starts)

  3. StatusChange (to “Playing”; stay here until enough players connect)

  4. StatusChange (to “PreGame”, once enough players have connected)

  5. Reset

  6. StatusChange (to “Playing”–the real game round now begins)

  7. (play game, beginning with players spawning in)

  8. StatusChange (to “EndGame”)

  9. DropKit (for each player still in game at end)
