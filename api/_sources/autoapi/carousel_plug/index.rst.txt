.. py:class:: Carousel(n_last_played=N_LAST_PLAYED, n_last_categories=N_LAST_CATEGORIES)



   Map rotation, picking random maps with filtering

   New maps are rolled during the *start* of each round, so the next map
   can be overridden by the ``!setnext`` command.

   The layer (size) of a map is chosen at the **end** of a map, since
   during a round, alot of players might (dis-)connect

   :Parameters:

       **n_last_played: int**
           number of last played maps to use in mapname filter

       **n_last_categories: int**
           number of last maps to use in category filter












   :Attributes:

       **n_connected: int**
           ..

       **last_maps: list(str)**
           names of last maps

       **last_cats: list(list(str))**
           list of category-lists from last maps

   .. method:: random_map(self)

      Pick a random nextmap

      Map start:
      - roll a map
      - reroll if duplicate theatre
      - reroll if not supporting current playernumber
      - pick layer based on current player num

      Map end:
      - pick new layer based on endgame playernum
      - if not possible, keep previous


      :Returns:

          int, str, int, str
              idx, mapname, layer, gamemode
